Ao baixar o Git, vc usara o Git Bash como terminal extendido para otimizar o uso do Git


Alguns comandos básicos para ter um bom desempenho no terminal do windows:
-Mudar de pastas: Utilizamos o "cd" e o nome da pasta para entrar em uma pasta expecífica por exemplo "cd windows", com isso vc estara entrando na pasta windows. Se utilizarmos o "cd .." voltaremos para a pasta anterior, ou seja se estavamos na base do disco local C usando o comando "cd windows" nos iriamos para a psta windows e usando o "cd .." voltariamos para a base do disco local C.
-Listar as pastas: Utilizamos o "dir".
-Criar pastas/arquivos: Utilizamos o "mkdir" e o nome da pasta ou arquivo que quer criar.
-Deletar pastas/arquivos: Utilizamos o "rmdir" e o nome da pasta ou arquivo que quer deletar.
-Limpar a tela: Utilizase o "cls" (Clear Screen) no Git Bash utilizasse o "CTRL + L".
